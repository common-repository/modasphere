Modasphere is official plugin developed by Casting Networks LLC for Modasphere's clients, who would like to use Modasphere functionality on their web sites. This plugin is bridge between Modasphere and Wordpress systems. Its main feature is creating web site menu with talents divisions and displaying full information about talent galleries and profiles.

== Installation ==

1. Download Modasphere's plugin on the PC
2. Upload the `Modasphere` folder to the `/wp-content/plugins` directory
3. Activate the plugin through the 'Plugins' menu in WordPress. "Modasphere" menu item is created in the admin menu.
4. Click on the "Modasphere" menu item.
5. Fill in all fields of "AUTH" section, click on "Save" button.
6. Go to "VISIBILITY OF MENU ITEMS" section and choose divisions which will be added to the site menu. Click on "Save" button.
7. In addition user can change params of talent photos in the "TALENT LIST SETTINGS" section. Options for displaying of the talents galleries, and items from talent profiles could be changed in the "TALENT SETTINGS" section.
8. Click on the 'house' icon in the left corner and check all applied changes on the website
